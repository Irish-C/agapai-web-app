# AGAPAI
AGAPAI is a web application designed for real-time monitoring and management of security cameras. 
It provides a user-friendly interface to view live camera feeds, manage camera settings, and receive alerts.

## Features
- Real-time camera feed display
- User authentication and role management
- Camera configuration and settings
- Alert notifications for motion detection
- Responsive design for desktop
- Integration with backend services via RESTful APIs
- WebSocket support for real-time updates

## Technologies Used
- React.js for the frontend
- Node.js and Express for the backend
- Socket.IO for real-time communication
- MySQL for database management
- CSS3 for styling and layout
- Vite as the build tool
- React Router for client-side routing
- React Icons for iconography
- Axios for HTTP requests
- Socket.IO Client for WebSocket communication
